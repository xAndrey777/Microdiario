# ✍️ One Line Diary

Un diario minimalista diseñado para capturar el mejor momento del día en una sola frase. Construido con HTML, CSS y JS puro.

## ✨ Características
- **Persistencia Local**: Tus datos se guardan en el navegador (`localStorage`).
- **Diseño Responsive**: Funciona bien en móviles y escritorio.
- **Enfoque en Simplicidad**: Sin cuentas, sin distracciones.

## 🛠️ Tecnologías
- HTML5
- CSS3 (Variables y Animaciones)
- JavaScript Vanilla

## 🚀 Instalación
Solo clona el repositorio y abre `index.html` en tu navegador.
